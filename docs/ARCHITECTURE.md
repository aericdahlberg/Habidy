## Current Data Model

### `users`
```sql
id                  uuid PRIMARY KEY
email               text UNIQUE NOT NULL
name                text
identity_statement  text        -- "I want to be someone who..."
goal_category       text        -- health | career | relationships | creativity | mindset | other
friction_point      text        -- "One thing that doesn't match who I want to be"
time_available      text        -- 5min | 15min | 30min | 60min+
new_user            boolean DEFAULT true   -- set to false after /welcome is viewed
onboarding_done     boolean DEFAULT false
created_at          timestamptz DEFAULT now()
```

### `habits`
```sql
id                  uuid PRIMARY KEY
user_id             uuid REFERENCES users NOT NULL
habit_name          text NOT NULL
identity_label      text        -- "I am a ___" — generated by Architect
cue                 text        -- "After I make coffee, I will..."
craving             text        -- optional
action              text        -- optional
reward              text        -- optional
two_minute_version  text
category            text        -- Health & Fitness | Career & Learning | etc.
color               text        -- hex, optional
time_of_day         text DEFAULT 'anytime'
is_active           boolean DEFAULT true
created_at          timestamptz DEFAULT now()
```

### `habit_logs`
```sql
id          uuid PRIMARY KEY
habit_id    uuid REFERENCES habits NOT NULL
user_id     uuid REFERENCES users NOT NULL
date        text NOT NULL   -- ISO date string
completed   boolean NOT NULL
logged_at   timestamptz DEFAULT now()
```

### `habit_survey_responses`
```sql
id                uuid PRIMARY KEY
user_id           uuid REFERENCES users NOT NULL
habit_id          uuid REFERENCES habits NOT NULL
date              text NOT NULL
went_right        text
went_wrong        text
completion_level  text   -- 'full' | 'partial' | 'none'
created_at        timestamptz DEFAULT now()
```

### `proposed_habits`
```sql
id          uuid PRIMARY KEY
user_id     uuid REFERENCES users NOT NULL
habit_data  jsonb NOT NULL   -- full habit object from Architect
selected    boolean DEFAULT false
created_at  timestamptz DEFAULT now()
```

### `conversation_memory`
```sql
id          uuid PRIMARY KEY
user_id     uuid REFERENCES users NOT NULL
agent       text NOT NULL   -- 'identity-gatherer' | 'architect'
summary     text            -- JSON + recap merged into one object
created_at  timestamptz DEFAULT now()
```

### `user_reflections`
```sql
id          uuid PRIMARY KEY
user_id     uuid REFERENCES users NOT NULL
content     text NOT NULL
created_at  timestamptz DEFAULT now()
```

### `user_profile_context`
```sql
id          uuid PRIMARY KEY
user_id     uuid REFERENCES users NOT NULL UNIQUE
summary     text            -- running AI-generated profile summary
updated_at  timestamptz DEFAULT now()
```

### `tool_logs`
```sql
id          uuid PRIMARY KEY
user_id     uuid
agent       text
tool_name   text            -- tool name or 'session_end'
input       jsonb
output      jsonb
success     boolean
error       text
duration_ms int
created_at  timestamptz DEFAULT now()
```

## Current API Routes

```
POST  /api/onboarding               -- save identity, goal_category, friction_point, time_available

GET   /api/habits                   -- get user's active habits (is_active = true)
POST  /api/habits                   -- create habit(s) — accepts single object OR array
POST  /api/habits/[id]/log          -- log completion or skip for today (upserts)
GET   /api/habits/[id]/streak       -- returns current streak + last 7 days
POST  /api/habits/survey            -- save habit_survey_responses row

POST  /api/agents/constellation     -- Identity Gatherer agent (POST messages, get reply)
POST  /api/agents/architect         -- Architect agent (POST messages, get reply)

GET   /api/memory/[agent]           -- get latest conversation_memory for this agent
POST  /api/memory/[agent]           -- save session summary

POST  /api/explore                  -- save user_reflections row + regenerate profile context
```

## Agent Guard Layer

Every agent call goes through the guard wrapper in `lib/agentGuard.ts`.

```typescript
const result = await agentGuard({
  agentName: 'identity-gatherer',
  toolName: 'getUser',
  input: { userId },
  fn: () => supabase.from('users').select('*').eq('id', userId).single(),
  assert: (result) => {
    if (!result.data) throw new Error('User not found')
  }
})
```

Rules:
1. Log every tool call and its return value — not just errors
2. Assert on tool outputs before passing to model — empty = stop
3. All logs go to `tool_logs` table
4. Run eval sets on every deploy — see `evals/`
